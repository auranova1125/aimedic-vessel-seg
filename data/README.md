# Dataset Layout

This directory contains the RCA angiography images and the annotation files
used by the training and inference scripts. Image data is intentionally kept
outside the repository; the empty directories below define the expected layout.

```text
data/
├── RCA_train/
│   ├── image/             # Raw angiography frames used for training
│   ├── label_r/           # Red vessel annotations
│   ├── label_b_g/         # Blue endpoint and green entrance annotations
│   ├── image_processed/   # 256x256 images generated by the preparation script
│   └── gt/                # Class-index masks generated by the preparation script
└── RCA_test/
    └── image/             # Angiography frames used for inference
```

## File Correspondence

Training files use the same filename stem across the related directories. For
example, `image/frame_001.jpg`, `label_r/frame_001.bmp`, and
`label_b_g/frame_001.bmp` describe one frame. The preparation script writes
`image_processed/frame_001.bmp` and `gt/frame_001.bmp` for that frame.

The annotation colors are interpreted as follows:

- Red: vessel
- Blue: vessel endpoint
- Green: vessel entrance

The generated ground-truth mask uses these class indices:

- `0`: background
- `1`: vessel without an endpoint or entrance label
- `2`: vessel endpoint
- `3`: vessel entrance

Endpoint and entrance labels are assigned only where a vessel label is also
present. Images and annotations may use `.bmp`, `.jpg`, or `.png` extensions.

## Preparation and Inference Paths

Run the preparation step from the project root after placing the raw training
files in `RCA_train/image`, `RCA_train/label_r`, and `RCA_train/label_b_g`:

```bash
python training/prepare_rca_dataset.py --train-dir data/RCA_train
```

The training script then reads `RCA_train/image_processed` and `RCA_train/gt`.
By default, the inference script reads test frames from `RCA_test/image`:

```bash
python training/predict_rca_unet.py \
  --input-dir data/RCA_test/image \
  --output-dir testing/outputs/RCA \
  --checkpoints-path training/checks/RCA/
```

Use `--input-dir` when the test images are stored in a different directory.
